# Helm Push

## Overview

Package and push Helm charts to GitHub Container Registry (GHCR) as an OCI registry as part of CI/CD.

## When to Use

- Adding a new Helm chart to CI/CD
- Modifying existing Helm chart configurations
- Understanding Helm versioning strategy
- Configuring OCI registry pushes
- Investigating Helm packaging issues
